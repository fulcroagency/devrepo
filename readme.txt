== Description ==

Dev Options WordPress plugin that helps you manage and optimize various aspects of your website. It is designed to work with a variety of features, divided into five main sections for better organization:

- General
- Security/Performance
- SEO/Marketing
- Other
- Plugins

This plugin is designed to help you enhance and manage your site's security, performance, SEO, and much more. Some features require the use of the GeneratePress theme or its child theme.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/dev-options` directory, or install the plugin through the WordPress plugin screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Begin configuring the available options through the plugin's settings page.

== Features ==

**General**
- Provides general management features to optimize your site's functionality and settings.

**Security/Performance**
- Enhance your site's security.
- Improve performance by optimizing certain areas.

**SEO/Marketing**
- Tools to improve your website's SEO and marketing efforts.

**Other**
- Additional features to enhance your WordPress experience.

**Plugins**
- Manage and configure plugins more easily.

== Requirements ==

- WordPress 5.0 or higher
- PHP 7.4 or higher
- GeneratePress theme (some features require the GeneratePress theme or its child theme)

== Frequently Asked Questions ==

= Do I need to use GeneratePress theme? =
Some features of the Dev Options plugin require the GeneratePress theme or a child theme based on GeneratePress. If you do not use GeneratePress, certain features will not be available.

= What version of PHP is required? =
Dev Options requires PHP version 7.4 or higher.
